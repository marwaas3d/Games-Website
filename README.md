## 📌 Description
Game Arena is a web-based platform that provides a collection of interactive mini games that run directly in the browser without any installation.

## ✨ Features
- Multiple mini games
- Fast and responsive design
- Simple and clean UI
- Works on all devices

## 🛠️ Tech Stack
- HTML
- CSS / SCSS
- bootstrap
- Vercel
